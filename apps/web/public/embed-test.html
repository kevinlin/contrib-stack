ContribStack Embed Test

This page simulates a third-party website embedding the ContribStack widget. It loads /widget.js and mounts two <contrib-stack> instances — the same way an external blog or portfolio would.

The host page applies aggressive CSS (color: red, font-family: Comic Sans MS) to every element. If Shadow DOM isolation works, the widgets should still render with their own theme colors and typography.
